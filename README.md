# Regression-Model-Comparison
Comparing of multiple regression models and selecting the best one 
## Overview
This project is a case study focused on implementing, comparing, and evaluating multiple regression models to predict a continuous target variable using Python.
This analysis is performed entirely in a Jupyter NoteBook
## Regression Models
--Linear Regression
--Ridge Regression
--Lasso Regression
--ElasticNet Regression
--Support Vector Regression (SVR)
--Decision Tree Regression
--Random Forest Regression
--Bayesian Linear Regression
 ## WorkFlow
1.Data loading and inspection
2. Feature and target separation
3. Train-test split
4. Feature scaling using StandardScaler
5. Model training
6. Performance evaluation
7. Best model comparison
